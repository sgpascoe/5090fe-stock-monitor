# Firebase Android Setup - Step by Step
# =====================================

## You're at Step 2: Download and add config file

### What to do RIGHT NOW:

1. **Download google-services.json**
   ✓ You've downloaded it from Firebase Console
   
2. **Place it in your Android project:**
   ```
   YourAndroidProject/
   └── app/
       └── google-services.json  ← Place it HERE
   ```

3. **If using Android Studio:**
   - Open your Android project
   - Drag google-services.json into the `app/` folder
   - Android Studio will ask to sync - click "Sync Now"

### Step 3: Add Firebase SDK

**In your app/build.gradle file, add:**

```gradle
// At the TOP of the file
apply plugin: 'com.android.application'
apply plugin: 'com.google.gms.google-services'  // ← Add this line

android {
    // ... your existing config
}

dependencies {
    // Add these Firebase dependencies:
    implementation 'com.google.firebase:firebase-messaging:23.4.0'
    implementation 'com.google.firebase:firebase-analytics:21.5.0'
}
```

**In your project-level build.gradle (root build.gradle), add:**

```gradle
buildscript {
    repositories {
        google()
        mavenCentral()
    }
    dependencies {
        classpath 'com.android.tools.build:gradle:8.1.0'
        classpath 'com.google.gms:google-services:4.4.0'  // ← Add this
    }
}

allprojects {
    repositories {
        google()
        mavenCentral()
    }
}
```

### Step 4: Sync Gradle

- Click "Sync Now" in Android Studio
- Or run: `./gradlew build`

### Step 5: Create the App Code

Copy the code from FCM_SETUP.txt:
- MainActivity.java
- MyFirebaseMessagingService.java  
- AndroidManifest.xml updates
- activity_main.xml layout

### Step 6: Run the App

1. Connect your Android phone
2. Enable USB debugging
3. Run the app
4. Copy the FCM token displayed on screen
5. Add to GitHub Secrets as FCM_DEVICE_TOKEN
