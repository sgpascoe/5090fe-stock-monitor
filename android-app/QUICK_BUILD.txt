# QUICK ANDROID APP BUILD GUIDE
===============================

FASTEST WAY - Use Android Studio:

1. Open Android Studio
2. File → New → New Project
3. Choose "Empty Activity"
4. Name: RTX Stock Alert
5. Package: com.rtxstock.alert
6. Language: Java
7. Minimum SDK: API 21 (Android 5.0)

8. Copy files:
   - Copy MainActivity.java → app/src/main/java/com/rtxstock/alert/
   - Copy MyFirebaseMessagingService.java → same folder
   - Copy AndroidManifest.xml → app/src/main/
   - Copy activity_main.xml → app/src/main/res/layout/
   - Copy google-services.json → app/

9. Add Firebase:
   - File → Project Structure → Dependencies
   - Add: com.google.firebase:firebase-messaging:23.0.0
   - Sync Gradle

10. Build & Run:
    - Connect your Android phone
    - Enable USB debugging
    - Click Run button
    - App will show your FCM token - COPY IT!

ALTERNATIVE - Use Online Builder:
----------------------------------
1. Go to: https://appinventor.mit.edu/
2. Create simple app that displays FCM token
3. Export APK and install

The app just needs to:
- Display FCM token
- Receive notifications
- That's it!

Your token will be displayed on screen - copy it and add as FCM_DEVICE_TOKEN secret!
